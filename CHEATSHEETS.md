# Cheat Sheets

## fzf

Most useful integrations and quick use cases.

### Fuzzy completion

Fuzzy completion for files and directories can be triggered if the word before
the cursor ends with the trigger sequence, which is by default **.

```
vim **<TAB>
```

### Shell key bindings (fzf key-bindings)

- Ctrl-T: fuzzy pick file(s) into the command line.
- Ctrl-R: fuzzy search shell history.
- Esc-C: fuzzy cd into a directory.
